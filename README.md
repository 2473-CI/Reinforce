## Run
运行前请确保具备以下环境

- nodejs >= 18
- jdk >= 17
- 推荐使用pnpm而非npm

### Server
```
cd Server
```
### Browser
```
cd browser

pnpm install # 或者 npm install 

pnpm run dev # 或者 npm run dev
```

### App
```
cd app

pnpm install # 或者 npm install 

pnpm dev:h5 # 或者 npm dev:h5
```
